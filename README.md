Compilador 2026.1

Projeto acadêmico de um compilador desenvolvido em Java.

📌 Sobre

Este projeto implementa um compilador dividido em etapas.
Atualmente, contempla a interface gráfica e o analisador léxico, responsável por identificar tokens e detectar erros no código-fonte da linguagem 2026.1.
